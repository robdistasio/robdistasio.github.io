Brief Instructions:

The Ensemble drop-down menu allows one to change which statistical ensemble the system is in. The Potential drop-down menu controls how the particles interact with each other. The Gravity Slider allows one to add in the effect of gravity on the particles, the added gravity in this simulation is very strong, where the maximum allowed gravitational force is over a billion times as strong as Earth's gravity! The Simulation Speed slider controls how fast the simulation runs.

Extra features such as a dividing wall can be added. When a dividing wall is present, all atoms added will generate on the left side of the wall. The permeable membrane reflects Argon and Xenon atoms while allowing Helium and Neon atoms to pass through freely.

Running the simulation:

First, let the system equilibrate and then hit the "Reset stats" button to start getting data from the equilibrium system. Note that adding or removing an atom will automatically reset the statistics.

Data can be written into the data area using the "Write Data" button. This will also write the data into a text file. Alternatively, the "Auto interval:" drop-down menu allows you to set a frequency at which the program will automatically write the data into your text file. Switching to "all atoms" on the Data type drop-down menu will make the Write Data button record the positions and velocities of all atoms in the system. Finally, to download the text file containing your data, hit the "Download File" button.
